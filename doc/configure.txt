INTRODUCTION                                    *configure*

Plugin for quickly configuring Vim.

                                                *<leader>ev*

                                                *<leader>sv*
<leader>sv                      Source .vimrc

                                                *<leader>egv*
<leader>egv                     Edit .gvimrc

                                                *<leader>sgv*
<leader>sgv                     Source .gvimrc

                                                *<leader>s*
<leader>s                       Source current buffer
